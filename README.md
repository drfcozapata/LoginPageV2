# Tracking Premium

This project was created with Vite 3.1.0 and:

- React 18.2.0
- Fluent UI 9.4.0
- React Router 6.4.2
- Redux Toolkit 1.8.6
- Axios 1.1.2
- i18next 21.10.0

## Install

To install dependencies, run the command

`yarn`

## Run Project

To run the project (local server), run the command

`yarn dev`
